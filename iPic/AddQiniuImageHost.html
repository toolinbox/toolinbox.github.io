在 iPic 中添加七牛云

发表于 2016-05-23   |   分类于 iPic   |  

七牛云是 iPic 在内测期间呼声比较高的国内图床，这里介绍下在 iPic 中添加七牛云的过程。

注册七牛账户

这点比较简单，直接到 这里 注册即可。

在七牛中添加资源

七牛的数据处理是以 资源 为单位的，这里我们可以简单理解为七牛中的 文件夹。

登录七牛后，进入 添加资源 页面，选择添加 对象存储。

这里我创建一个 test 资源。


在 iPic 中添加七牛图床

打开 偏好设置，进入 图床 页，选择添加 七牛。以下分别介绍各项的意义及如何配置：

  • 名称
    • 主要是方便自己区分多个图床，可以任意取名，比如 我的七牛
  • 资源
    • 就是刚才创建的 资源，这里即是 test
  • AccessKey 与 SecretKey
    • 可以简单理解为七牛的隐身账户和密码
    • 进入 个人面板 > 密钥管理页
    • 依次将 AK、SK 中的内容复制到 iPic 对应的 AccessKey、SecretKey 中
  • 网址前缀
    • 用于生成上传图片的完成 Http 路径
    • 打开刚刚创建的 test 资源，复制 测试域名 中的域名至 iPic 的 Http 前缀中。要注意格式的完整，比如 http://o7entp---.bkt.clouddn.com/ 就是格式完整的 Http 前缀。
    • 当然，如果你绑定了域名，就可以使用自己的域名。
    • 另外，如果需要使用七牛的文件名前缀功能（也可理解为文件夹），只要在 Http 前缀最后附上文件名前缀即可。
      • 比如，普通的 Http 前缀是：http://o7entpabc.bkt.clouddn.com/；
      • 添加 test 文件名前缀后即变为：http://o7entpabc.bkt.clouddn.com/test/；
      • 上传的图片链接类似于http://o7entpabc.bkt.clouddn.com/test/screenshot.jpg
  • 网址后缀
    • 可用于自定义图片样式。比如，在七牛中添加了名字为 s 的样式表示缩略图，样式分隔符为 -，则可以在 网址后缀 中输入 -s。这样之后每次上传图片生成的链接最后都会附上 -s，也即表示缩略图。

完成输入后，可以点击 验证 按钮。如果输入没有问题，则右侧会出现 通过 链接，点击后就可以看到测试时上传至七牛的图片。

最后，点击 应用 按钮进行保存。

使用七牛

在 iPic 菜单 > 图床 中选择刚刚添加的七牛，之后上传的图片就会在七牛图床中。

注：本文中的图就是通过上述方式先上传至七牛、再粘贴至 Markdown 编辑器中，确实很方便，大家试试吧。
